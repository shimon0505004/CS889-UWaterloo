Solution Description

The solution shows three scatterplots side by side to show corresponding points for PCA, MDS and t-SNE reduction. These reductions convert the high-dimentional data points into 2D points. The dataset contains 10 categories of data (10 different labels). Each label is colorcoded on the top of the corresponding scatterplot. The X and Y axis correspond to the 2D values generated by the dimentionality reduction.

With the full dataset, the MDS and t-SNE reductions take too much running time. To resolve this issue, we limited the number of iterations that MDS and t-SNE reductions can run. We limited t-SNE to 250 iterations and MDS reduction to 20 iterations for the data to be generated quickly.

From the visible data, we see that in PCA and TSNE reduction, the reduced data is forming labelwise clusters. In PCA reduction, the clusters are distributed around the axis center.

In PCA reduction, the clusters form a skewed, sheared elipse. From the plot it looks like the major axis for these elipsis clusters intersect around a point (-1600, -460) approximately. The cluster boundaries are mostly well-defined for label class 1, 3, 0, 9, 8. Other cluster areas overlap with each other in a way that their boundaries superimpose.

In TSNE reduction the label clusters are markedly seperated in different axis. Label 5, Label 9 and Label 7 do not go beyond Q3 quadrant and all other clusters are mostly situated in Q1 quadrant. Meaning, with this reduction it is possible to determine probable supergroup of the label [group (5,9,7) or group (1,2,3,4,6,8,0)] with reasonable accuracy once we have the reduced feature vectors.

In case of MDS reduction, we generated the data using a iteration of 1 and 20. In the first iteration, the projected data was randomly scattered around the axes, but with the increase of iteration in the MDS reduction method, a cluster pattern started to emmerge. In the MDS scatterplot, we observed similar label based ring shaped clusters centered around the center of the axis. We believe with increased iteration, the ring clusters would form better in a way that boundaries between different label clusters could be clearly established, and data could be classified with reasonable accuracy given the reduced feature vectors.
